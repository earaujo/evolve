# Evolve

> Static project page based on Jekyll

**Requirements**

 * Ruby 1.9+ + `$ gem install jekyll`
 * Bower (Node.js + NPM) + `$ bower install`

To test the web page locally, run `$ jekyll serve` and open the browser in http://localhost:400/evolve/.
